# 引易 (YinYi)

> 引易，取易经之"易"，寓意使引用转换变得简单。

一个优雅简洁的在线工具，致力于简化法学论文引用格式的转换工作。

## 使用说明
1. 选择转换方向：
   - 法学引注手册 → GB/T7714-2015
   - GB/T7714-2015 → 法学引注手册
2. 在输入框中粘贴需要转换的引用文本
3. 点击转换按钮
4. 复制转换结果

## 功能特点 (MVP)
- 支持《法学引注手册》格式与 GB/T7714-2015 国标格式的双向转换
- 支持三种文献类型转换：
  - 图书
  - 期刊论文
  - 学位论文
  - 报纸
  - 网络文献
- 纯前端实现，无需后端服务
- 简洁直观的用户界面

## 技术架构
- HTML5
- CSS3
- Vanilla JavaScript
- 正则表达式进行格式识别和转换
